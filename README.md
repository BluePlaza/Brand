## ロゴについて
### 配色
青 (#0043aa)：技術的・冷静さを表現

白 (#ffffff)：潔白・シンプルさを象徴

### ロゴ構成
背景の白いアーチ・・・
広場を表し下の直線は大地を表します。BとPを包括する形で穏やかさを表現しています。

BとP・・・
下の直線（大地）から芽生える抽象的な形で、これから成長していくことや基盤をしっかり築くことを目指すイメージです。
